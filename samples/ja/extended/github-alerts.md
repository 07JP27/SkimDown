# GitHub アラート

GitHub アラートは重要な情報を目立たせるための特別な引用ブロックです。SkimDown は 5 種類すべてのアラートに対応しています。

## Note（補足）

> [!NOTE]
> 流し読みしていても知っておくべき、役立つ情報です。

## Tip（ヒント）

> [!TIP]
> より良く、より簡単に物事を進めるための役立つアドバイスです。

## Important（重要）

> [!IMPORTANT]
> 目的を達成するために知っておく必要がある重要な情報です。

## Warning（警告）

> [!WARNING]
> 問題を避けるため、すぐに注意が必要な緊急情報です。

## Caution（注意）

> [!CAUTION]
> 特定の操作によるリスクや悪影響についての注意喚起です。

## リッチコンテンツを含むアラート

> [!NOTE]
> アラート内では **太字**、*斜体*、`インラインコード` などの書式が使えます。
>
> 複数の段落やリスト、その他の要素も含められます：
>
> - 項目 1
> - 項目 2
> - 項目 3
>
> ```swift
> print("アラート内でもコードブロックが使えます！")
> ```

## リンクを含むアラート

> [!TIP]
> 詳しくは [SkimDown リポジトリ](https://github.com/07JP27/SkimDown) をご覧ください。
>
> 画像やその他のインライン要素も含められます。

## 通常の引用ブロック（アラートではない）

> これは通常の引用ブロックで、アラートではありません。
> 標準の引用ブロックスタイルで表示されるはずです。
