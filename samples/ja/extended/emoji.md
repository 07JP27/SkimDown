# 絵文字ショートコード

SkimDown は GitHub スタイルの絵文字ショートコードに対応しています。コロン `:` で絵文字名を囲んで入力します。

## 顔と感情

:smile: :laughing: :blush: :heart_eyes: :sunglasses:

:cry: :sob: :angry: :scream: :thinking:

:wink: :stuck_out_tongue: :zany_face: :shushing_face: :nerd_face:

## ジェスチャー

:pray: :muscle: :point_right: :ok_hand: :v:
