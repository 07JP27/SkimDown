## このファイルの意味

サイドバーのツリー表示では、フォルダは展開/折りたたみが可能です。この深い階層のファイルを表示することで:

1. ツリーのインデントが正しく表示されるか
2. フォルダの展開/折りたたみが機能するか
3. 深い階層からの相対リンクが正しく解決されるか

を確認できます。

> 📂 `deep/` → 📂 `nested/` → 📂 `folder/` → 📄 `deep-file.md`
