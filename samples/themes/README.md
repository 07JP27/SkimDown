# SkimDown Sample Themes

VS Code-format color theme JSONs you can drop into
`~/Library/Application Support/SkimDown/Themes/` to try out the custom color
scheme feature.

Copy a file into the Themes folder, then choose **View → Theme → Reload Themes**
and pick it from the same submenu.

See [`docs/usage/color-schemes.md`](../../docs/usage/color-schemes.md) for the
full color key mapping.
