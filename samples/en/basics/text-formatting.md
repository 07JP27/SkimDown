# Text Formatting

## Bold

**Bold text** is wrapped with two asterisks or two underscores.

__This is also bold__ text.

## Italic

*Italic text* is wrapped with one asterisk or one underscore.

_This is also italic_ text.

## Bold + Italic

***Bold and italic*** text is wrapped with three asterisks.

___This is also bold and italic___ text.

## Strikethrough

~~Strikethrough~~ text is wrapped with two tildes.

~~Old information~~ → Updated with new information.

## Inline Code

Wrap with a single backtick like `console.log("Hello")`.

Use `make build` to run the build command. The file path is `/usr/local/bin`.

## Combinations

Text formatting can be combined:

- **Text with *italic* inside bold**
- *Text with **bold** inside italic*
- ~~Text with **bold** inside strikethrough~~
- **`Bold inline code`** is also possible
