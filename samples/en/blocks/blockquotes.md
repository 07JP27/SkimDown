# Blockquotes

## Basic Blockquote

> This is a blockquote. It's used when quoting part of a text.

## Multi-line Blockquote

> Blockquotes can span multiple lines.
> Simply add `>` at the beginning of each line.
>
> You can also separate paragraphs with a blank line.

## Nested Blockquotes

> First level blockquote.
>
> > Second level blockquote. You can nest quotes within quotes.
> >
> > > Third level blockquote. You can nest even deeper.

## Formatting Inside Blockquotes

> You can include **bold text** and *italic text* inside blockquotes.
>
> `Inline code` works too.
>
> ~~Strikethrough~~ is also possible.

## Lists Inside Blockquotes

> To-do list:
>
> 1. Complete the design
> 2. Implement it
> 3. Test it
>
> - Note A
> - Note B

## Code Inside Blockquotes

> Hello World in Swift:
>
> ```swift
> print("Hello, World!")
> ```
