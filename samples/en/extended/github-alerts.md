# GitHub Alerts

GitHub Alerts are special blockquotes that highlight important information. SkimDown supports all five alert types.

## Note

> [!NOTE]
> Useful information that users should know, even when skimming content.

## Tip

> [!TIP]
> Helpful advice for doing things better or more easily.

## Important

> [!IMPORTANT]
> Key information users need to know to achieve their goal.

## Warning

> [!WARNING]
> Urgent info that needs immediate user attention to avoid problems.

## Caution

> [!CAUTION]
> Advises about risks or negative outcomes of certain actions.

## Alerts with Rich Content

> [!NOTE]
> Alerts can contain **bold**, *italic*, `inline code`, and other formatting.
>
> They can also have multiple paragraphs, lists, and more:
>
> - Item one
> - Item two
> - Item three
>
> ```swift
> print("Code blocks work inside alerts too!")
> ```

## Alerts with Links

> [!TIP]
> Check out the [SkimDown repository](https://github.com/07JP27/SkimDown) for more details.
>
> You can also include images and other inline elements.
