# Color Codes

SkimDown shows a small color swatch after hexadecimal color codes in prose.

- Brand primary: #0a66d6
- Success: #1a7f37
- Warning: #d29922
- Short hex: #f80
- With alpha: #8250dfcc

Inline examples work inside sentences, such as using #fbfbfd for a light background and #0f1116 for a dark background.

Color codes inside inline code, such as `#cf222e`, remain plain code.
