# Settings

Open the Settings window with `SkimDown > Settings…` or `Cmd+,`.

## General

- **Sidebar Position** — switch the sidebar between left and right.
- **Clear Recent Folders** — remove all entries from `File > Open Recent`.

## Themes

- **Theme** — choose System, Light, or Dark. The System option follows the macOS appearance.

## Editing

- **Font** — click "Choose Font…" to open the macOS font panel and select a custom body font. Click "Use Default" to return to the system font.
- **Font Size** — drag the slider to adjust the preview font size between 11 pt and 28 pt. Click "Reset" to restore the default 16 pt.

## Notes

- Changes apply immediately to all open windows.
- Theme and Zoom options in the View menu continue to work alongside the Settings window.
- Settings are persisted across app restarts.
