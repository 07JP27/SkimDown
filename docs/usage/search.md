# Search

SkimDown searches the Markdown file currently displayed in the preview.

## Shortcuts

- `Cmd+F`: show the search bar
- `Enter`: next match
- `Shift+Enter`: previous match
- `Esc`: close the search bar
- `Cmd+G`: find next
- `Shift+Cmd+G`: find previous
- `Cmd+E`: use the current selection for search

## Case sensitivity

Use the Case checkbox in the search bar to toggle case-sensitive search. The setting is saved.

## Scope

Search is intentionally limited to the current Markdown file. File-name search and cross-file search are future extensions, not MVP features.
