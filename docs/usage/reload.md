# Live reload

SkimDown watches the opened folder for Markdown changes.

## Tree updates

When Markdown files are added, removed, or renamed, SkimDown rescans the folder and updates the sidebar tree.

## Preview updates

When the currently displayed Markdown file changes, SkimDown reloads the preview automatically. If the displayed file is deleted, the window returns to an empty state.

## Image changes

Changes to images referenced from Markdown are reflected the next time the Markdown preview is rendered.
