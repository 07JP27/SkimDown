# インストール

SkimDown はネイティブ macOS アプリです。パッケージ済みリリースはまだ公開していません。

## 必要環境

- macOS 26以降
- `.md` または `.markdown` ファイルを含むフォルダ

## アプリを入手する

リリースパッケージは、用意でき次第 GitHub Releases から配布します。

## 初回起動

1. SkimDown を開きます。
2. **Open Folder...** を選びます。
3. 読みたい Markdown ファイルが入っているフォルダを選択します。

SkimDown は macOS のフォルダ選択ダイアログを使ってアクセスします。選んだフォルダを読み取り、その後も最近開いたフォルダとして再表示できるように権限を保存します。

## アップデート

パッケージ済みリリースが公開された後は、新しいビルドをダウンロードし、Applications フォルダ内の古いアプリを置き換えてください。

## 次に読む

起動後の操作は [使い方](./usage.md) を参照してください。フォルダを開く、ツリーを移動する、Markdownをプレビューする、表示中ファイルを検索する流れを説明しています。
