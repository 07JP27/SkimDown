# プライバシーと権限

SkimDown はローカルの読み取り専用 Markdown ビューアーです。選択したフォルダを読むためのアプリであり、ファイルを管理したり変更したりするためのアプリではありません。

## フォルダアクセス

SkimDown は macOS のフォルダ選択ダイアログで、ユーザーが明示的に選んだフォルダだけを読み取ります。

最近開いたフォルダや前回開いたフォルダを自動で再表示できるよう、選んだフォルダを macOS のファイル bookmark として保存します。bookmark はフォルダの移動やリネームを追従するため、ファイルシステムを整理した後でも履歴は引き続き動作します。

## 読み取り専用

SkimDown は Markdown ファイルを編集、保存、エクスポート、印刷、変更しません。アプリ内に書き込みのコードパスは存在しません。読むことと移動することに絞った設計です。

## ローカルファイル境界

相対 Markdown リンクやローカル画像は、解決先が開いているフォルダ内に留まる場合だけ扱います。Markdown ファイルを使ってフォルダ外の任意のローカルファイルを閲覧することはできません。

## HTML サニタイズ

埋め込み HTML は表示前にサニタイズします。危険なタグ、イベント属性、危険な URL スキームは除去します。

## 外部コンテンツ

外部リンクは既定のブラウザで開きます。Markdown から参照されている外部画像はプレビュー内で読み込まれる場合があります。

## 外部への送信なし

SkimDown は Markdown 本文やフォルダ内容を外部サービスへ送信しません。外部リンクはブラウザで開き、外部画像は上記の通りプレビュー内で読み込まれる場合があります。
