# 検索

SkimDownは、現在表示しているMarkdownファイル内だけを検索します。

## ショートカット

- `Cmd+F`: 検索バーを表示
- `Enter`: 次の一致へ
- `Shift+Enter`: 前の一致へ
- `Esc`: 検索バーを閉じる
- `Cmd+G`: 次を検索
- `Shift+Cmd+G`: 前を検索
- `Cmd+E`: 選択中テキストを検索に使う

## 大文字小文字

検索バーのCaseチェックボックスで、大文字小文字を区別するか切り替えられます。この設定は保存されます。

## 検索範囲

検索対象は表示中のMarkdownだけです。ファイル名検索や複数ファイル横断検索は将来拡張で、MVPには含めません。
