# 設定

Settingsウィンドウは `SkimDown > Settings…` または `Cmd+,` で開きます。

## General

- **サイドバー位置** — サイドバーを左右に切り替えます。
- **最近開いたフォルダのクリア** — `File > Open Recent` のすべてのエントリを削除します。

## Themes

- **テーマ** — System、Light、Dark から選択します。System は macOS の外観設定に追従します。

## Editing

- **フォント** — 「Choose Font…」をクリックすると macOS のフォントパネルが開き、本文フォントを変更できます。「Use Default」でシステム既定に戻ります。
- **フォントサイズ** — スライダーでプレビューのフォントサイズを 11 pt から 28 pt の範囲で調整します。「Reset」で既定の 16 pt に戻ります。

## 補足

- 変更はすべての開いているウィンドウに即座に反映されます。
- View メニューの Theme や Zoom 操作も引き続き使用できます。
- 設定はアプリ再起動後も保持されます。
