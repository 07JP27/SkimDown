# ファイルを開く

SkimDownはフォルダを開かなくても、単一のMarkdownファイルをプレビューできます。単一ファイルモードではサイドバーが非表示になり、プレビューのみが表示されます。

## 開き方

次の方法でファイルを開けます。

- `File > Open File...`
- `Shift+Cmd+O`
- 空のSkimDownウィンドウへ `.md` / `.markdown` ファイルをドラッグ&ドロップ
- Finderでファイルを右クリックし `このアプリケーションで開く > SkimDown` を選択
- コマンドライン: ファイルパスを引数として渡す（下記参照）

すでにフォルダを開いているウィンドウへMarkdownファイルをドロップした場合、新しいウィンドウで開きます。

## 単一ファイルモードの動作

- サイドバーは非表示になり、切り替えできません。
- ウィンドウタイトルにファイル名が表示されます。
- ファイルの親ディレクトリ内の相対画像パスやアセットリンクは通常通り解決されます。他のMarkdownファイルへのリンクは遷移しません。複数ファイル間のナビゲーションにはフォルダモードを使用してください。
- ファイル変更を監視し、プレビューを自動更新します。
- 単一ファイルウィンドウにフォルダをドロップすると、フォルダモードに切り替わります。

## コマンドライン

```sh
# 特定のファイルを開く
skimdown /path/to/README.md
```

Markdownファイルパスを引数に指定すると、前回のセッション復元をスキップして単一ファイルモードで開きます。
