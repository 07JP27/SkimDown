# フォルダを開く

SkimDownは1ウィンドウにつき1フォルダを読みます。各ウィンドウはフォルダ、選択ファイル、サイドバー状態、プレビュー状態を持ちます。

## 開き方

次の方法でフォルダを開けます。

- `File > Open Folder...`
- `Cmd+O`
- 空状態の `Open Folder...` ボタン
- 空のSkimDownウィンドウへのフォルダドラッグ&ドロップ
- コマンドライン: フォルダパスを引数として渡す（下記参照）

すでにフォルダを開いているウィンドウへ別フォルダをドロップした場合、新しいウィンドウで開きます。

## 初期選択

フォルダを開いた直後は、次の順で表示するMarkdownを決めます。

1. そのフォルダで前回開いていたMarkdown
2. `README.md`
3. ツリー内の先頭Markdown
4. Markdownがない場合は空状態

## 最近開いたフォルダ

最近開いたフォルダは `File > Open Recent` から開けます。可能な場合はsecurity-scoped bookmarkでフォルダ権限を復元します。

## コマンドライン

`skimdown` コマンドを使うには、アプリバイナリへのシンボリックリンクを作成してください。

```sh
ln -s /Applications/SkimDown.app/Contents/MacOS/SkimDown /usr/local/bin/skimdown
```

ターミナルからSkimDownを起動できます。

```sh
# 特定のフォルダを開く
skimdown /xxx/yyy/zzz

# カレントフォルダを開く
skimdown
```

フォルダパスを引数に指定すると、前回のセッション復元をスキップしてそのフォルダを直接開きます。引数なしでターミナルから起動した場合は、カレントディレクトリを開きます。カレントディレクトリがルート（`/`）の場合は、通常のセッション復元にフォールバックします。
